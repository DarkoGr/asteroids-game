# Asteroids Game 🚀

Arcade-style game inspired by the classic Asteroids, built in Python using Pygame.

## Features
- Player movement and rotation
- Shooting mechanics
- Asteroid spawning and splitting
- Collision detection system
- Score tracking

## Tech Stack
- Python
- Pygame

## Project Structure
- main.py – entry point
- player.py – player logic
- asteroid.py / asteroidfield.py – asteroid behavior
- shot.py – shooting system
- explosion.py – effects

## How to run

```bash
pip install pygame
python main.py
